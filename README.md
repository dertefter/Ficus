# Ficus

Неофициальный клиент личного кабинета для студентов НГТУ (НЭТИ)

Функционал приложения:
🔸 Расписание занятий и расписание сессии
🔸 Зачётка
🔸 Сообщения от преподавателей и служб
🔸 Новостная лента прямо из университета
🔸 Интеграция с DiSpace (курсы, диалоги, расписание вебинаров)
🔸 Информация о стипендиях и выплатах
🔸 Расписание преподавателей
🔸 Читательский билет
🔸 Запись в бюро пропусков
🔸 Поделиться успеваемостью
🔸 Поддержка Material You и цветов Monet (Для Android 12+)
