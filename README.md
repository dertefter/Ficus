# Ficus

Неофициальный клиент личного кабинета для студентов НГТУ (НЭТИ)

Функционал приложения:</br>
🔸 Расписание занятий и расписание сессии</br>
🔸 Зачётка</br>
🔸 Сообщения от преподавателей и служб</br>
🔸 Новостная лента прямо из университета</br>
🔸 Интеграция с DiSpace (курсы, диалоги, расписание вебинаров)</br>
🔸 Информация о стипендиях и выплатах</br>
🔸 Расписание преподавателей</br>
🔸 Читательский билет</br>
🔸 Запись в бюро пропусков</br>
🔸 Поделиться успеваемостью</br>
🔸 Поддержка Material You и цветов Monet (Для Android 12+)</br>
